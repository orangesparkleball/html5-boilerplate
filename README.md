#  HTML5 Boilerplate [http://html5boilerplate.com](http://html5boilerplate.com)

OSB HTML5 Boilerplate - html5boilerplate adapted to use staticmatic.

To use: 

    $ bundle install
    $ bundle exec staticmatic preview .
    ... test changes ...
    $ bundle exec staticmatic build .
